What actions can the operator take? ▾
📨 Send parent advisory
Auto-drafted EN / UR / PA text dispatched via SMS, WhatsApp, school PA. Operator edits + approves before send. Template library covers heatwave, smog, flood watch, cold snap, routine.
🕐 Modify school day
Early dismissal, advanced pickup time, no outdoor activity, double water rations, indoor PE only, cancel assembly. Marks school day as adjusted in the log.
🏥 Notify health responders
BHU / nearest clinic flagged that a child-impact event is active at school location. Heat-stroke / asthma kits requested if not on site.
🚫 Recommend closure
Drafts a closure recommendation for district admin sign-off. System never closes a school autonomously — that's a government call.
⏱ Defer 24h
Acknowledge but defer action one day (forecast may improve). Alert reappears in tomorrow's queue if conditions persist.
📋 Mark as monitoring
No advisory yet — operator is watching the situation. Audit-logged so reviewers know the alert wasn't simply ignored.

Privacy
The advisory engine never stores parent PII. Operator-side gateway holds contacts; this system holds only school IDs, hazard scores, and dispatch logs. Parents can opt out per channel; opt-outs are honoured immediately.
Open-source note
The template library and content engine ship as open source. Operators plug in their own SMS / WhatsApp Business / e-bell gateway. We don't lock partners into our messaging stack.

How the hub works — one paragraph

Every day, five open climate data sources are pulled for each school's coordinates and stored in a single open data layer (CC BY 4.0). A scoring engine compares each school's current readings to its 10-year baseline and outputs heat / air-quality / flood vulnerability scores plus a child-burden estimate. When a score crosses a threshold the system drafts a multilingual parent advisory and surfaces it for operator approval in Hazard Watch. The operator dispatches via their existing SMS / WhatsApp / PA gateway; the school confirms what action was taken; the event is logged for audit and future ML.
