School Climate Hub

Open infrastructure for school-level climate exposure data, and operational tools to act on it for children's health.

Founding deployment: 50 PSSP/PSRP schools, ~12,000 students with PDLC in Gujranwala, Pakistan.

Open operator dashboard demo → View source on GitHub
For school operators: apply for verified access → · For parents: subscribe to alerts for your child's school (coming Q1 2027)

Vulnerability dashboard

Per-school heat, air quality, flood, and rainfall scores from open satellite + reanalysis data — ERA5, MODIS, Sentinel-5P, CAMS, GloFAS. EMIS-keyed.

Multilingual advisories

Auto-drafted parent advisories in English, Urdu, and Punjabi (Shahmukhi). Operator approves; messages dispatch via existing SMS / WhatsApp / school PA channels.

Open data layer

A clean, school-level, EMIS-keyed climate exposure dataset other operators can build on. CC BY 4.0. Daily refresh. Mirrors to HDX + Zenodo.
